# Weather App

A responsive weather application that lets users search for the current weather in any city. Built as part of **The Odin Project** to practice API integration and JavaScript fundamentals.

 
📂 https://github.com/SIAMECALEB/Weather-App

## 📌 About the Project
This weather application uses a public weather API (such as OpenWeatherMap) to fetch real-time weather data based on user search input.  
The app demonstrates fetching data, handling user input, and dynamically updating the UI.

This project was built to practice:
- Fetching data from APIs
- JavaScript logic and ES6 features
- DOM manipulation
- Error handling
- Responsive UI design

## 🛠️ Built With
- HTML5  
- CSS3  
- JavaScript  
- API integration (OpenWeatherMap or similar)  
- Git & GitHub

## ✨ Features
- Search weather by city name
- Display weather conditions, temperature, humidity, etc.
- Responsive layout for mobile & desktop
- Clean and user-friendly interface

## 📚 What I Learned
- Fetching and handling data from APIs
- Asynchronous JavaScript (`fetch`, promises)
- Updating the UI dynamically
- Managing user input and error handling
- Project organization and structure
